# Amr Fakhri Portfolio

A premium single-page portfolio website for showcasing UX/UI design work.

## Live setup with GitHub Pages
1. Create a new GitHub repo (example: `amr-portfolio`)
2. Upload all files in this folder to the repo root
3. Go to **Settings → Pages**
4. Under **Build and deployment**, choose:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / `(root)`
5. Save and wait a minute

Your site will be live at:
`https://YOUR_USERNAME.github.io/amr-portfolio/`

## Tech

- Vanilla JavaScript
- GSAP + ScrollTrigger
